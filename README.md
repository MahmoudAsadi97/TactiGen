# TactiGen

Evidence-Grounded Multimodal AI System for Football Tactical Analysis and Action Anticipation.

## Status
🚧 Under active development — built by automated agent.

## What this system does
TactiGen transforms short football video clips (15–30 seconds) into structured tactical evidence,
predicted future actions, visual analytics, and grounded coaching reports.

## Tech Stack
Python · YOLOv8 · PyTorch Transformer · FAISS · SentenceTransformers · GPT-4o · Streamlit · PostgreSQL
